# 2.4 桁架平衡方程组LDLT分解求解作业
## 项目说明
基于桁架刚度组装代码，自主实现对称正定矩阵LDLT分解求解器；
包含矩阵分解、三步求解、奇异矩阵识别、耗时统计、残差精度计算、病态矩阵测试功能。

## 文件清单
1. truss_global_stiffness_24.py：主求解程序，修改case_id切换一维/二维桁架
2. case2_2d.json：二维桁架模型输入文件
3. *.png：程序运行控制台截图
4. *.pdf：完整作业报告（理论、算例、附录源码）

## 运行说明
依赖numpy；运行代码，case_id=1一维桁架，case_id=2二维桁架；桁架计算完成自动运行三组数值测试矩阵。
