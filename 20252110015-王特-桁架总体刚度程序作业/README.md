# 桁架总体刚度矩阵有限元程序作业
## 文件清单
1. truss_global_stiffness.py：一维/二维桁架通用有限元求解主程序
2. case2_2d.json：二维桁架算例模型输入文件
3. 一维两单元杆运行结果.png：case_id=1控制台完整输出截图
4. 二维桁架完整运行截图.png：case_id=2控制台完整输出截图
5. 作业报告.pdf：完整理论推导、算例结果分析

## 运行依赖
Python3 + numpy
安装命令：pip install numpy

## 使用方法
1. 修改代码末尾 case_id：
case_id=1 运行一维两杆验证算例
case_id=2 运行二维平面桁架算例
2. 运行程序后控制台输出：刚度矩阵、节点位移、支座反力、单元应力轴力
